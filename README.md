# Agent Knowledge Book · Agent 知识大全

面向「软件研发工程师转型 Agent 研发」的系统性知识库。

## 项目定位

本项目是一部 Agent 领域的知识大全，覆盖从历史沿革到当下前沿技术的完整知识体系。目标读者为有软件工程背景、正在或即将转型 Agent 方向的工程师。

## 内容结构

知识大全分为两大部分：

**Part I · Agent 史学** — 从 1950s 早期 AI 到 LLM Agent 时代的发展脉络，重点关注 LLM-based Agent 的兴起与迭代，提炼人类开发 Agent 的历史经验与教训。

**Part II · Agent 技术体系** — 对当下最先进的 Agent 技术进行系统性拆解，包括架构模式、内部模块组织、Multi-Agent 协作、评测体系，以及主流框架/产品的全景分类。

## 语言约定

- 正文以中文撰写
- 英文术语首次出现时标注原文，后续可直接使用
- 历史资料保留原始引用（类似论文引用格式）
- 代码示例、配置、CLI 命令保留英文

## 如何使用

按目录结构浏览 `docs/` 目录下的 Markdown 文件树即可。每个章节自成体系，也可通过顶层目录索引进行跳转。
