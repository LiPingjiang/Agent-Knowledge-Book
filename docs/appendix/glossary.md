# 术语表

Agent 领域核心术语的中英对照与简要释义。

| 术语 | 英文 | 释义 |
|------|------|------|
| 智能体 | Agent | 能够感知环境并采取行动以实现目标的自主实体 |
| 大语言模型 | Large Language Model (LLM) | 基于 Transformer 的大规模预训练语言模型 |
| 思维链 | Chain-of-Thought (CoT) | 让模型逐步推理的提示技术 |
| 工具调用 | Tool Use / Function Calling | Agent 调用外部工具或 API 的能力 |
| 多智能体系统 | Multi-Agent System (MAS) | 多个 Agent 协作完成任务的系统 |
