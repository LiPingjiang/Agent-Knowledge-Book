# LLM Agent 崛起

2022-2023年，大型语言模型的能力涌现彻底改变了Agent的构建方式。从思维链推理到ReAct范式，从工具使用到自主Agent，LLM为Agent赋予了前所未有的通用推理和自然语言交互能力。这一时期见证了从学术探索到全民热潮的快速转变。
