# 前 LLM 时代

在大型语言模型出现之前，Agent 的研究已经走过了半个多世纪的历程。从早期的符号AI和专家系统，到BDI架构、多智能体系统，再到强化学习Agent，每一个阶段都为今天的 LLM Agent 奠定了理论和工程基础。

## 核心脉络

这五个阶段并非简单的线性替代关系，而是从不同角度探索了"如何构建智能体"这一核心问题：

- **符号 AI** 回答了"智能体需要什么能力"——规划、知识、推理
- **BDI 与反应式** 回答了"智能体应该如何决策"——审慎与反应的平衡
- **多智能体系统** 回答了"多个智能体如何协作"——通信、协商、协调
- **强化学习** 回答了"智能体如何自主学习"——试错、奖励、策略优化
- **对话系统** 回答了"智能体如何与人交互"——语言理解与生成

LLM Agent 的突破在于，它首次将这些能力统一在一个系统中：用语言模型同时实现规划、推理、学习、通信和交互。
